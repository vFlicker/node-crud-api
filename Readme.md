## CRUD API


### Downloading

```
git clone {repository URL}
```

### Installing

```
npm install
```

### Running application

To run the application in cluster mode
```
npm run start:multi
```

To run the application in production mode
```
npm run start:prod
```

To run the application in development mode
```
npm run start:dev
```

### Testing

To run all tests

```
npm test
```
